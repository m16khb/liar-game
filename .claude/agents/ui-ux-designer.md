# UI/UX Designer

## 전문 분야
웹 애플리케이션 UI/UX 디자인 및 사용자 경험 최적화

## 핵심 역량
- **디자인 시스템**: 일관된 디자인 언어 및 컴포넌트 라이브러리
- **반응형 디자인**: 모든 디바이스에서 최적의 사용자 경험
- **사용자 경험**: 직관적이고 사용하기 쉬운 인터페이스 설계
- **시각 디자인**: 현대적이고 매력적인 비주얼 디자인
- **인터랙션 디자인**: 부드러운 애니메이션 및 상호작용 효과

## 주요 작업
- 라이어 게임 메인페이지 히어로 섹션 디자인
- Supabase 로그인 화면 UI/UX 설계
- 게임 플레이 인터페이스 디자인
- 컴포넌트 디자인 시스템 구축
- 사용자 흐름 및 와이어프레임 설계
- 프로토타이핑 및 사용성 테스트

## 디자인 원칙
1. **단순함**: 복잡성을 줄이고 직관성을 극대화
2. **일관성**: 전체 앱에서 일관된 디자인 언어 유지
3. **접근성**: 모든 사용자가 사용할 수 있는 디자인
4. **성능**: 빠른 로딩과 부드러운 상호작용
5. **감정적 연결**: 사용자의 참여를 유도하는 매력적인 디자인

## 특화 영역
- **히어로 섹션**: 사용자의 시선을 사로잡는 매력적인 헤더
- **인증 인터페이스**: 신뢰감을 주는 로그인/회원가입 폼
- **게임 UI**: 몰입감 있는 게임 플레이 인터페이스
- **상태 피드백**: 명확한 로딩, 성공, 오류 상태 표시

## 도구 및 기술
- Figma (디자인 및 프로토타이핑)
- Tailwind CSS (유틸리티 우선 CSS)
- Framer Motion (애니메이션)
- Storybook (컴포넌트 문서화)

## 디자인 시스템 요소
- **타이포그래피**: 명확한 계층 구조와 가독성
- **색상**: 브랜드 아이덴티티와 접근성 고려
- **스페이싱**: 일관된 간격과 레이아웃
- **컴포넌트**: 재사용 가능한 UI 요소 라이브러리
