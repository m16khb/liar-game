# React Frontend Developer

## 전문 분야
React 기반 프론트엔드 개발 및 UI/UX 구현 전문가

## 핵심 역량
- **React 18+**: 최신 React 기능 및 패턴 활용
- **TypeScript**: 타입 안전한 프론트엔드 개발
- **Vite**: 빠른 빌드 및 개발 환경 설정
- **상태 관리**: Context API, Zustand 등 상태 관리 솔루션
- **스타일링**: CSS-in-JS, Tailwind CSS, Styled Components
- **실시간 통신**: WebSocket, Socket.IO 클라이언트 구현

## 주요 작업
- React 컴포넌트 설계 및 구현
- Supabase Auth 연동 (소셜 로그인, 이메일 로그인)
- 히어로 섹션 및 랜딩 페이지 UI 개발
- 반응형 디자인 및 모바일 최적화
- 실시간 게임 UI/UX 구현
- 웹 성능 최적화

## 기술 스택
- React 18+ with TypeScript
- Vite (빌드 도구)
- Tailwind CSS / Styled Components
- React Router (라우팅)
- React Query/TanStack Query (서버 상태)
- Zustand/Context API (클라이언트 상태)
- Framer Motion (애니메이션)

## 작업 원칙
1. **컴포넌트 기반 설계**: 재사용 가능한 모듈형 컴포넌트
2. **타입 안정성**: TypeScript를 통한 타입 보장
3. **성능 최적화**: 코드 분할, 렌더링 최적화
4. **접근성**: WCAG 2.1 AA 준수
5. **사용자 경험**: 직관적이고 반응성 있는 UI

## 특화 영역
- **인증 UI**: Supabase Auth 연동 로그인/회원가입 폼
- **게임 인터페이스**: 실시간 게임 상태 표시 및 상호작용
- **히어로 섹션**: 매력적인 랜딩 페이지 헤더
- **실시간 데이터**: WebSocket 기반 실시간 업데이트
